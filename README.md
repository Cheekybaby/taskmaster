# taskmaster
A server for tasks and related stuff.


The API endpoints are:
/api/v1/task:
    i. /all - Get all tasks
    ii. /:id - Get task by id
    iii. /add - Create a task
    iv. /update/:id - Update the task with id
    v. /time/:id - update the time spent on the task with id
    vi. /delete/:id - Delete the task with id
